P-4 Source Point Website

The Lockheed Martin P-3 Source Point website was commisioned by the defense contractor to facilitate owners and operators of the P-3 reconnissance aircraft in requisitioning parts for their planes. The website was created by my company, Organizational Strategies Inc, and put into production in October of 2014. The website allows users to request orders for parts from Lockheed Martin's licensed distributors and manufacturers. It also provides allows users to open up issues regarding part numbers that are not documents or out of production. There is also a discussion area for the site where users can discuss alternative strategies for securing parts, trading parts, coordinating joint purchases to leverage economies of scale, and knowledge sharing on P-3 operation and maintenance. The website project begane in July of 2014 and was completed three months later with a team of three developers utilizing a Scrum methodology. The website runs on JBoss powered by Java, Mysql, and HTML. Other technologies used in the website include the Activiti Workflow Engine, Spring, CVS, Apache Ant, and OSI's proprietary Inter Domain Service Manager, which provides database connectivity, user validation, session handling, and a GUI framework.

Android Netrunner

The Android: Netrunner Arena Limited website is a personal project I created to facilitate Limited play for Fantasy Flight Games' Android: Netrunner Card Game. A user is able to construct a deck list consisting of one of 3 randomly chosen idendities and 30 cards chosen one at a time from a pool of three cards created according to a specified configuration. This project was created in my spare time for personal use. It uses HTML, Google App Engine, and Mysql. Other technologies involved include Bootstrap Front End Framework, Javascript, Jquery, Ajax, Hibernate, and github.
